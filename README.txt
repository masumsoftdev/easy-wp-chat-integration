Custom Phone Call and WhatsApp Chat Button for website.

== Description ==
This plugin enables a feature of WhatsApp and Phone call button integration easily to website.

== Installation ==
1. Upload the `Easy WP Chat Integration` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the WordPress Plugins Page.
3. Navigate to the WordPress Admin Menu \'Easy WP Chat\'. 
4. Enter your Numbers.
5. Save your settings, and you\'re all set!
